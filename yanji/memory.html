自定义指令 & 跨窗口记忆

这里写全局指令、管理每个会话的摘要，以后可以做成真正的“跨窗口大脑”。

🤖 自动记忆

开启后，AI 会自动从对话中提取值得记住的信息，无需手动添加。

提取频率（每隔几轮对话） 3

数字越小提取越频繁，但也会消耗更多 token。

🧠 自然记忆（RAG）

像人脑一样的记忆：自动形成，只在相关时浮现，常用的记得牢，不用的慢慢淡忘。
需要：① 登录 Supabase ② 执行数据库设置脚本 ③ 有 OpenAI 或 Gemini API

已形成的记忆

这些记忆会在相关话题时自然浮现，不会机械地列举。

还没有自然记忆，聊几轮后会自动形成。
📝 记忆条目（传统）

一条条添加你希望AI记住的信息，可以随时编辑、删除、开关。发请求时会自动拼到提示词里。

还没有记忆条目，点下方按钮添加。
生成参数（全局，对所有模型生效）

调一调下面几个滑块，可以控制话多话少、敢不敢跑题、回得长不长。

温度（temperature） 0.7
最大回复长度（max_tokens） 4096
重复惩罚（frequency_penalty） 0
话题新鲜度（presence_penalty） 0
上下文限制（滑动窗口）

限制发送给模型的历史对话长度，省钱又避免超限。超出部分会自动从最早的对话开始丢弃。

保留最近对话轮数 50
一轮 = 你发一条 + AI回一条
最大上下文 Token 数 30000
粗略估算：1000 tokens ≈ 500-600 中文字
选择要编辑摘要的会话

通常选最近在用的几个窗口，给它们写一句话简介，就像你给每条时间线写“目录”。
